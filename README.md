# JProg
A programming language made in Java

## Planned Features
 - Variables Handling
 - Scope Handling
 - Memory Management
 - Functions
 - Loops
 - Conditionals

## Requirements
 - Java 23
 - Gradle 8.10.2
 - JUnit Jupiter 5.10.3
 - Guava 33.2.1-jre

## Project Structure
 - `src/main/java/com/jprog/`: Main source code
 - `src/test/java/com/jprog/`: Test source code
